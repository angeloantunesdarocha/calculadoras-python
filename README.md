# Calculadoras em Python 🖩

Calculadoras e operações matemáticas em Python, desenvolvidas durante os estudos de **Sistemas de Informação** no IFNMG.

## 📚 Sobre o repositório

Implementações de calculadoras simples e menus interativos para operações matemáticas básicas.

## 📁 Arquivos

| Arquivo | Descrição |
|---|---|
| `Dois_valores_e_4_operacoes.py` | Calcula as 4 operações básicas entre dois valores |
| `Menu_de_quatro_operacoes_matematica.py` | Menu interativo com soma, subtração, multiplicação e divisão |

## 🛠️ Tecnologias

- **Python 3.x**

## ▶️ Como executar

```bash
git clone https://github.com/angeloantunesdarocha/calculadoras-python.git
cd calculadoras-python
python3 Menu_de_quatro_operacoes_matematica.py
```
